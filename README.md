# Merchant

Merchant is a full-stack collectibles market tracker for researching, tracking, and analyzing prices across categories like trading cards, watches, vinyl records, and other collectible assets.

## Features

- Browse collectible market data by category
- Track historical price trends
- View market analytics and category insights
- Manage watchlist and portfolio-style views
- Use backend API routes for collectibles, categories, prices, and market data
- Built with a database-backed architecture for real market-tracking workflows

## Tech Stack

- React
- TypeScript
- Express
- PostgreSQL
- Drizzle ORM
- Tailwind CSS
- Chart.js
- Vite

## Run Locally

```bash
npm install
npm run dev
```

This project uses PostgreSQL for database-backed features. Environment variables are listed in `.env.example`.

## Project Status

Merchant is a portfolio project focused on full-stack architecture, market data modeling, dashboard UI, and price-tracking workflows.
